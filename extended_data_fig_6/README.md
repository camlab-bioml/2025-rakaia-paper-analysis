# Extended Data Figure 6

The code and files here can reproduce Extended Data Figure 6 of the rakaia paper.

The raw IMC image and single-cell segmentation mask for the ROI shown in this figure can be obtained here: https://idr.openmicroscopy.org/webclient/img_detail/10502265/?dataset=11851

The file `Ali-BC_MB0653_1_63_cell_types.csv` can be imported into Rakaia to generate the cell-type classification projections as shown in sub-panels B, C, and D; these classifications are found as `cluster` in the file. 

With both the raw full stack image and segmentation mask imported, Rakaia can perform quantification of the segmented cells to re-create the gating for panels C and D. Information on generating measurements for gating in Rakaia can be found here: https://rakaia.io/docs/quantification#quantification
